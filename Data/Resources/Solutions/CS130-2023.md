1) For each part of the question, find an undirected simple graph $G = (V, E)$ with the specified properties, or prove that none exists.


**(a)** $G$ is a tree in which no vertex has degree $1$.

>[!hint]- Hint 1
> Recall the definition: a tree is a **connected, acyclic** graph. The definition says nothing about minimum degree.

>[!hint]- Hint 2
> Does a tree have to have every vertex of degree $\geq 1$? Think about the *smallest* tree you can possibly write down.

>[!check]- Solution
> Yes, take the trivial graph $G = (\{v\}, \emptyset)$, a single vertex with no edges.
>
> $G$ is connected (trivially), acyclic (no edges, hence no cycles), so it is a tree. Its only vertex has degree $0$, which is not $1$, so no vertex of $G$ has degree $1$.
>
> *Note: for any tree with $\geq 2$ vertices, a standard counting argument shows the tree has at least $2$ leaves (vertices of degree $1$). So this single-vertex tree is essentially the only example.*


**(b)** $G$ is a tree which has $5$ connected components and $|V| = 10$.

>[!hint]- Hint 1
> Recall the definition: a tree is a **connected, acyclic** graph. How many connected components does a connected graph have?

>[!check]- Solution
> No such graph exists.
>
> A tree is connected by definition, so it has exactly **one** connected component. A graph with $5$ connected components cannot be a tree.


**(c)** $G$ is non-planar, $|V| = 10$, and $|E| = 8$.

>[!hint]- Hint 1
> Recall **Euler's formula** for connected planar graphs: $|V| - |E| + |F| = 2$, where $|F|$ is the number of faces.

>[!hint]- Hint 2
> $|F| \geq 1$ for any planar embedding: every embedding has at least one face, the unbounded outer face.

>[!check]- Solution
> No such graph exists.
>
> By Euler's formula for connected planar graphs, $|V| - |E| + |F| = 2$. Substituting $|V| = 10$ and $|E| = 8$:
> $|V| - |E| + |F| = 10 - 8 + |F| = 2 + |F|$.
>
> Since $|F| \geq 1$ (any planar embedding has at least one face: the unbounded outer face), $2 + |F| \geq 3 > 2$, contradicting Euler's $|V| - |E| + |F| = 2$. Hence no such graph exists.


**(d)** $G$ has $5$ connected components, $|V| = 10$, and $|E| = 9$.

>[!hint]- Hint 1
> The minimum number of edges needed to have $C$ connected components on $V$ vertices is $V - C$ (a spanning forest of $C$ trees). Anything beyond that goes into making one or more components "thicker" than a tree.

>[!hint]- Hint 2
> With $V = 10$ and $C = 5$, you need at least $V - C = 5$ edges. You have $9$ to spend, so $4$ edges go on top of a spanning forest. Try keeping most components as isolated vertices and putting all the edges into one component.

>[!check]- Solution
> Yes, take $4$ isolated vertices together with one copy of $K_{3,3}$ (the complete bipartite graph on $3 + 3 = 6$ vertices, which has $9$ edges).
>
> This gives $5$ connected components ($4$ singletons plus one $K_{3,3}$), $|V| = 4 + 6 = 10$, and $|E| = 0 + 9 = 9$.


**(e)** $G$ contains an Eulerian cycle, $|V| = 10$, and $|E| > 40$.

>[!hint]- Hint 1
> Recall: a connected graph has an Eulerian cycle iff every vertex has **even** degree.

>[!hint]- Hint 2
> The maximum number of edges in a simple graph on $10$ vertices is $\binom{10}{2} = 45$, achieved by $K_{10}$. To get $|E| > 40$, we need to remove fewer than $5$ edges from $K_{10}$. What does removing an edge do to the parity of each vertex's degree?

>[!check]- Solution
> No such graph exists.
>
> Start from $K_{10}$, which has $\binom{10}{2} = 45$ edges and every vertex of degree $9$ (odd). Any simple graph on $10$ vertices with $|E| > 40$ is obtained from $K_{10}$ by removing at most $4$ edges.
>
> Removing an edge $\{u, v\}$ decreases the degrees of both $u$ and $v$ by $1$, flipping their parities. To make every vertex of $G$ have even degree, we'd need to flip the parity of all $10$ vertices, so each vertex must be incident to an *odd* number (in particular, at least one) of the removed edges.
>
> By the handshake lemma applied to the *removal graph*, the sum of removal-degrees equals $2 \cdot |\text{removed}|$. Since every vertex has removal-degree $\geq 1$, the sum is $\geq 10$, so $|\text{removed}| \geq 5$.
>
> Hence $|E| \leq 45 - 5 = 40$, contradicting $|E| > 40$. So no such graph exists.
